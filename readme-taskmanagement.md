# Task Management README

## Introduction

This repository contains a simple task management application written in Python. It allows users to create, update, and delete tasks.

## Requirements

- Python 3.7+
- Flask 1.1.1+

## Installation

1. Clone the repository: `git clone https://github.com/example/task-management.git`
2. Install the required packages: `pip install -r requirements.txt`

## Usage

1. Navigate to the project directory: `cd task-management`
2. Run the application: `python app.py`
3. Open your web browser and go to `http://localhost:5000`

## Features

- Create tasks: Enter a title and description for the task and click the "Add Task" button.
- Update tasks: Click the "Edit" button next to a task to modify its details.
- Delete tasks: Click the "Delete" button next to a task to remove it from the list.
