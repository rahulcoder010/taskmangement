## Features

### This project has several features

- login & logout user auth
- Displaying a list of tasks with their title, description and status.
- Allowing users to update the status of tasks by interacting with the UI.
- Web Socket Integration
- To ensure real-time synchronization between clients, web socket integration is implemented. This allows any updates made by one user to be immediately reflected in the task list for all active clients. 
- For example, if a user updates the status of an existing task, those changes are immediately visible to all other connected clients.
